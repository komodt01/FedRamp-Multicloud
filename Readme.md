FedRAMP Multi-Cloud Security Architecture
This project implements a FedRAMP-aligned security framework in AWS and Azure, leveraging NIST 800-53 controls for compliance. 

*Note this is not meant to be a comprehensive security solution.  There has be an understanding the business requirements then align with the technology that meets thos requirements.  This is a home project*

📌 Features
✔ Multi-cloud FedRAMP security (AWS & Azure)
✔ Automated security enforcement using Terraform
✔ SIEM monitoring (AWS Security Hub & Azure Sentinel)
✔ Role-based access control (RBAC, IAM, Zero Trust)
✔ Logging, compliance, and incident response
